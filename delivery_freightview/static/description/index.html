Freightview Shipping

Provides streamline your shipment.

This module extends the functionality of Odoo v14 to streamline your shipment and increase your accuracy and efficiency. It helps you to compare, book, track and analyse shipment with your carriers.
You can see and compare negotiated freight and parcel rates in one place instead of going to a lot of different carrier websites.
It supports LTL, Partial truckloads, flatbed, etc..

Features

User can see the Carriers details.

User can see list of several transport details with quotes.

Easy to compare and book.

User can easily compare the quote and book their freight.

Tracking on Delivery Order.

User can track the shipment reference and cost in delivery order.

Vendor bill Reconciliation.

User can also reconcile for thier incomming shiping cost.
Video Tutorial

Lets see the screenshots for the overview of the module functionality.

Freightview Configuration.
Set the freightview API key details in "Shipping Method".
Sale Order.
When making a new quotation, the user has the option of selecting a freightview delivery method.
Delivery Order.
After confirmation of sale order user can see the create quote in freightview button in delivery order to connect freightview to get quotes.
Freightview quotes.
It lists the number of carriers details with cost user can choose their suitable carrier by thier own.
After choosing a quote, the customer can arrange their shipment for a time that is convenient for them.
Once the order is scheduled user can view the shipment number and tracking information.
Shipment info from freight view.
User can see the Get shipment info from freightview button on the Delivery order after the freightview was scheduled.
By clicking the Get shipment info from freightview shipping cost and tracking reference was automatically filled.
Vendor bill reconciliation.
By creating the vendor bill with the bill reference of the tracking number of the freightview shipment and user can also see the Check freightview button on bill.
Once click the Check freightview button it automatically fetch the freightview shipment link and Delivery or receipt information from the freightview and user can post the entry and register the payment
